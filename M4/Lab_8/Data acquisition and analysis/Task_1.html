DAQ Assistant

DAQ Assistant

Creates, edits, and runs tasks using NI-DAQmx. Refer to the NI-DAQmx Readme for a complete listing of devices NI-DAQmx supports.

When you place this Express VI on the block diagram, the DAQ Assistant launches to create a new task. After you create a task, you can double-click the DAQ Assistant Express VI to edit that task. For continuous measurement or generation, place a while loop around the DAQ Assistant Express VI.

Distortion Measurements

Distortion Measurements

Performs distortion measurements on a signal, such as tone analysis, total harmonic distortion (THD), and signal in noise and distortion (SINAD).

--------------------

This Express VI is configured as follows:

Measurement:Total Harmonic Distortion

Stop at Nyquist: Yes

Highest Harmonic: 10

Tone Measurements

Tone Measurements

Finds the single tone with the highest amplitude or searches a specified frequency range to find the single tone with the highest amplitude. You also can find the frequency and phase for a single tone.

--------------------

This Express VI is configured as follows:

Measurement: Amplitude

Measurement: Frequency

Approximate Frequency: 10

Search Tolerance: 5 %
